# Sales Performance Analysis & Forecasting

## Project Overview
This project analyzes 3 years of global sales data to identify business trends, customer behavior, regional performance, and future sales forecasting using Power BI.

## Tools & Technologies
- Power BI
- Power Query
- DAX

## Key Insights
- Analyzed 36,230 orders from 9,063 customers
- Generated total sales of 9M and profit of 3.97M
- United States contributed highest orders and profit
- Identified monthly growth trends with peak growth rate of 58.12%
- Created sales forecasting dashboard for future trend prediction

## Dashboard Features
- Sales KPI Cards
- Region-wise Performance Analysis
- Monthly Sales Trend
- Profit Analysis
- Forecasting Visualization
- Customer Insights
